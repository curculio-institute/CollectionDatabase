taxon

Local taxon / OTU. Stores rank-level fields from order down. dwc:scientificName is derived at export (genus + specificEpithet + authorship). parent_id enables tree navigation alongside the denormalised rank columns. dwc:taxonomicStatus: "accepted" | "synonym" | "invalid".

taxon_determination

Links a specimen to a taxon. Multiple rows per specimen preserve history. is_current = 1 marks the accepted determination. dwc:verbatimIdentification is the name exactly as written on the label.

biological_association

Joins a subject to an object via a relationship type. Subject and object are each either a collection_object or a taxon — enforced by two exclusive-arc CHECK constraints (exactly one of each pair must be non-null). Cannot be imported via DwC; local-master, no automated push. Verified: occurrence.rb:948 @897f385.
